Compliance LGPD

Última atualização: {% now "d/m/Y" %}

1. Compromisso com a LGPD

A RM Systems está comprometida em cumprir integralmente a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 - LGPD) e garantir que todos os dados pessoais tratados através do nosso Sistema sejam protegidos e utilizados de forma responsável e transparente.

2. Nossos Papéis sob a LGPD

2.1. Controlador de Dados

Como Controlador, determinamos os propósitos e os meios do tratamento de dados pessoais dos usuários do Sistema (dados de contas, informações de login).

2.2. Operador de Dados

Como Operador, processamos dados em nome das empresas clientes (dados dos mapas e análises), conforme instruções estabelecidas pelas empresas, que são os controladores desses dados.

3. Princípios de Tratamento de Dados

Seguimos rigorosamente os princípios estabelecidos pela LGPD:

  • Finalidade: Tratamos dados apenas para finalidades específicas, explícitas e legítimas
  • Adequação: O tratamento é compatível com as finalidades informadas
  • Necessidade: Limitamos o tratamento ao mínimo necessário para alcançar as finalidades
  • Livre Acesso: Garantimos transparência sobre o tratamento de dados
  • Qualidade dos Dados: Mantemos os dados atualizados e precisos
  • Transparência: Fornecemos informações claras e acessíveis sobre o tratamento
  • Segurança: Implementamos medidas técnicas e organizacionais para proteger os dados
  • Prevenção: Adotamos medidas para prevenir danos aos dados
  • Não Discriminação: Não realizamos tratamento discriminatório
  • Responsabilização: Demonstramos conformidade e adotamos medidas efetivas

4. Base Legal para o Tratamento

Tratamos dados pessoais com base nas seguintes bases legais previstas na LGPD:

  • Execução de Contrato: Para fornecer os serviços solicitados
  • Cumprimento de Obrigação Legal: Para cumprir obrigações legais e regulatórias
  • Legítimo Interesse: Para melhorar nossos serviços e segurança
  • Consentimento: Quando você nos dá consentimento explícito

5. Direitos dos Titulares de Dados

Conforme a LGPD, você possui os seguintes direitos em relação aos seus dados pessoais:

5.1. Direitos Garantidos

  • Confirmação e Acesso: Saber se tratamos seus dados e ter acesso a eles
  • Correção: Solicitar correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, Bloqueio ou Eliminação: Solicitar anonimização, bloqueio ou eliminação de dados desnecessários, excessivos ou tratados em desconformidade com a LGPD
  • Portabilidade: Solicitar portabilidade dos dados para outro fornecedor
  • Eliminação: Solicitar eliminação de dados tratados com base em consentimento
  • Informação sobre Compartilhamento: Obter informações sobre entidades públicas e privadas com as quais compartilhamos dados
  • Informação sobre Possibilidade de Não Consentir: Ser informado sobre a possibilidade de não fornecer consentimento e as consequências
  • Revogação do Consentimento: Revogar seu consentimento a qualquer momento

5.2. Como Exercer seus Direitos

Para exercer qualquer um dos direitos acima, entre em contato conosco através do sistema de suporte ou por e-mail. Responderemos sua solicitação no prazo de até 15 dias, conforme exigido pela LGPD.

6. Medidas de Segurança

Implementamos medidas técnicas e organizacionais adequadas para proteger os dados pessoais, incluindo:

  • Criptografia: Dados em trânsito (HTTPS/TLS) e em repouso
  • Controle de Acesso: Acesso restrito e baseado em função
  • Isolamento Multi-tenant: Isolamento completo de dados entre empresas
  • Monitoramento: Monitoramento contínuo de segurança
  • Backup: Backups regulares e seguros dos dados
  • Treinamento: Treinamento de pessoal sobre proteção de dados
  • Políticas de Segurança: Políticas e procedimentos documentados

7. Compartilhamento de Dados

Não vendemos dados pessoais. Compartilhamos dados apenas:

  • Com prestadores de serviços que nos ajudam a operar o Sistema (sujeitos a acordos de confidencialidade e proteção de dados)
  • Quando exigido por lei ou ordem judicial
  • Com seu consentimento explícito

Todos os prestadores de serviços que acessam dados pessoais são contratualmente obrigados a manter a mesma proteção que exigimos internamente.

8. Retenção de Dados

Mantemos dados pessoais apenas pelo tempo necessário para cumprir as finalidades para as quais foram coletados, salvo se um período de retenção mais longo for exigido ou permitido por lei. Quando os dados não forem mais necessários, os eliminamos de forma segura.

9. Notificação de Incidentes

Em caso de incidente de segurança que possa acarretar risco ou dano relevante para os titulares de dados, notificaremos a Autoridade Nacional de Proteção de Dados (ANPD) e os titulares afetados conforme exigido pela LGPD.

10. Encarregado de Proteção de Dados (DPO)

Para questões relacionadas à proteção de dados e aos seus direitos sob a LGPD, você pode entrar em contato com nosso Encarregado de Proteção de Dados através do sistema de suporte ou por e-mail.

11. Autoridade Nacional de Proteção de Dados (ANPD)

Se você acredita que seus dados pessoais foram tratados de forma inadequada, você tem o direito de apresentar uma reclamação à ANPD:

  • Website: www.gov.br/anpd
  • Ouvidoria: falabr.cgu.gov.br

12. Atualizações

Esta página pode ser atualizada periodicamente para refletir mudanças em nossas práticas de proteção de dados ou na legislação aplicável. Recomendamos que você revise esta página regularmente.

13. Contato

Para questões sobre proteção de dados e compliance com a LGPD, entre em contato conosco através do sistema de suporte ou por e-mail.

Para mais informações, consulte também nossa Política de Privacidade.
